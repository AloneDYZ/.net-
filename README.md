### 🚩 项目介绍

SmartSQL 是一款方便、快捷的数据库文档查询、导出工具！从最初仅支持`SqlServer`数据库、`CHM`文档格式开始，通过不断地探索开发、集思广益和不断改进，又陆续支持`Word`、`Excel`、`PDF`、`Html`、`Xml`、`Json`、`MarkDown`等文档格式的导出。同时又扩展支持包括`SqlServer`、`MySql`、`PostgreSQL`、`SQLite`等多种数据库的文档查询和导出功能。
